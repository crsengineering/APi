LIS3DSH breakout board using same (minimal) form factor of ADXL355-EVAL


LIS3DSH MEMS accelerometer




Pinout:
1 - VDDIO (power supply for I/O)
2 - VDD (power supply for MEMS sensor)
3 - GND
4 - DRDY (data ready output signal)
5 - SDO (serial data out for SPI interface)
6 - SDI (serial data in for SPI interface)
7 - SCK (serial clock for SPI interface)
8 - CS (chip select for SPI interface)
